***# Real-Time Hand Gesture Recognition System***



***## Description***

***This project implements a real-time hand gesture recognition system using OpenCV and MediaPipe.***



***## Features***

***- Hand gesture detection***

***- Finger counting***

***- Volume control using hand gestures***

***- Screenshot capture using thumbs up***

***- Real-time performance with FPS display***



***## Technologies Used***

***- Python***

***- OpenCV***

***- MediaPipe***

***- PyAutoGUI***



***## How to Run***

***1. Install dependencies:***

&#x20;  ***pip install opencv-python mediapipe pyautogui numpy***



***2. Run the program:***

&#x20;  ***python gesture.py***



***## Output***

***- Gesture detection in real-time***

***- System control using hand gestures***
